## Removing Branches Local

```
      $  git branch -D dev 
      $  git branch -D test
```
## Removing Branches REMOTE
After Deleting
```
        $ git push origin : dev 
        $ git push origin : test 
```

## Listing Tags 
```
        $ git tag -l
```
 

## Deleting  Tags 
```
        $ git tag -d v1.7
        $ git push origin --delete v1.7
```
